# Candela — Unified AI and Skill Exchange Platform

## Overview

Candela is a unified workspace designed to eliminate friction while working, learning, or troubleshooting. It merges multiple AI systems, community skill exchange, voice accessibility, and seamless utilities into one cohesive platform.

## Features

* **Candela AI (Unified LLM Switching):** Switch between ChatGPT, Gemini, and Perplexity within the same chat without re-entering prompts.
* **Skill Tradeoff System:** Post problems based on your weak areas and help others using your strengths, enabling community-driven problem solving.
* **Voice-Read Responses:** AI-generated outputs can be read aloud for accessibility and convenience.
* **Spotify Integration:** Listen to music while working without leaving the app.
* **Smooth and Intuitive UI:** Clean layout optimized for clarity and ease of navigation.
* **Secure Authentication:** Google Login with a protected database environment.

## Mission

To centralize intelligence, collaboration, and workflow into one platform so users never need to switch between apps for answers, help, or productivity.
